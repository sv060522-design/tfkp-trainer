# Пакеты исправлений задач

Каждый файл вида `vN-*.js` добавляет или уточняет задачи через общий объект `window.TFKP_TASK_OVERRIDES`.

Старый файл `data/tasks.js` хранит исходную OCR-базу и стабильные ID. Его лучше не трогать.

Новые проверенные условия и решения добавляются сюда отдельными пакетами. Это нужно, чтобы будущие обновления не стирали старые правки.

## v6-solutions.js

Крупный пакет подробных решений: 2008/2009 вариант 81, 2002/2003 вариант 22, дополнительные пояснения к задачам с вычетами, рядами Лорана, особыми точками и конформным отображением.
