# V5 major update

Что изменено:

- По умолчанию сайт открывает только проверенные задачи, чтобы OCR-мусор не попадался первым.
- Старые задачи не удалены: их можно включить фильтром «Все условия» или «Требуют сверки OCR».
- Добавлена пакетная система обновлений: новые правки лежат в `data/batches/v5-major.js`.
- Старые `id` задач не менялись, поэтому прогресс пользователей не должен слететь.
- Добавлены/расширены решения и ответы для крупных блоков:
  - 2021 вариант 1: задачи 1–6;
  - 2002/2003 осень, варианты 1–2: вычищенные условия, подробные решения для части задач, ответы для всех задач;
  - 2003/2004: вариант 31 оформлен подробнее, для вариантов 32–34 внесены сверенные ответы по Пырковой;
  - 2008/2009 вариант 81: вычищенные условия и ответы, подробные решения для задач 5–6;
  - 2014/2015 вариант 41 сохранён из предыдущего пакета.

## Как обновить сайт

1. Скопировать содержимое архива в локальную папку репозитория с заменой файлов.
2. Выполнить:

```powershell
git add .
git commit -m "major verified tfkp update v5"
git push
```

Vercel обновится автоматически.

## Как дальше добавлять v6 без перетирания старого

Для следующей партии нужно будет добавить новый файл:

```text
data/batches/v6-major.js
```

и добавить в `index.html` одну строку перед `app.js`:

```html
<script src="./data/batches/v6-major.js"></script>
```

Файл должен иметь вид:

```js
window.TFKP_TASK_OVERRIDES = Object.assign(window.TFKP_TASK_OVERRIDES || {}, {
  "task-id": {
    status: "verified",
    statementPretty: "...",
    solution: "...",
    answer: "..."
  }
});
```

Так старые пакеты не стираются, а новые просто дописывают и уточняют задачи.
