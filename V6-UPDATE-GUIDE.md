# v6 update guide

Пакет `v6-solutions.js` не перезаписывает исходную базу `data/tasks.js`.
Он добавляет новые подробные решения поверх уже установленного v5 через `Object.assign`.

## Как применить

1. Скопировать файлы из архива в папку репозитория с заменой.
2. Выполнить:

```powershell
git add .
git commit -m "add detailed solutions v6"
git push
```

Vercel сам обновит сайт.
