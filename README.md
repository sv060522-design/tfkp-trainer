# ТФКП — тренажёр · ФЭФМ МФТИ

Статический сайт-тренажёр по ТФКП. Его можно открыть локально двойным кликом по `index.html`, а в интернет выложить через GitHub + Vercel.

## Что внутри

- `494` карточек задач/отзывов.
- Семестровые варианты 1996/97–2014/15 и свежие файлы, которые были в папке `/mnt/data`.
- Реальные отзывы с экзамена 2019 из `ТФКП 2019 (Ответы).xlsx`.
- Ответы к варианту 2014/2015 №41 по твоему фото.
- Ответы к варианту 2008/2009 №81 и решения к задачам 5–6 по файлам Пырковой.
- Прогресс, звёздочки и отметки «решено» через `localStorage`.
- Экспорт/импорт прогресса JSON-файлом.

## Важное ограничение

Старые PDF местами извлекаются с плохим OCR/кодировкой. Такие задачи помечены `needs-review` или «сверить OCR». Перед тем как считать задачу финальной, нужно открыть исходный PDF и сверить формулу.

## Как добавлять задачи

Открой файл:

```text
data/tasks.js
```

Каждая задача — объект внутри массива `window.TFKP_TASKS`. Главное правило: **не менять старые `id`**, иначе у пользователей слетят отметки прогресса по этим задачам.

## Как запустить локально

Самый простой способ: открыть `index.html` в браузере.

Более правильно, через локальный сервер:

```bash
python -m http.server 5173
```

И открыть:

```text
http://localhost:5173
```
